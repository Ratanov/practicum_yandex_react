## Команды

### Установка

- `npm i` — установка стабильной версии

### Запуск и сборка

- `npm run dev` — запуск dev версии
- `npm run build` — сборка стабильной версии

### Тестирование и деплой

- `npm run test` — jest тестирование
- `npm run deploy` — деплой на GitHub Pages
